📜 <pre>

What it does: The <pre> tag preserves spaces, tabs, and line breaks exactly as written in the code.

Why use it: Perfect for writing code snippets or formatted text (like poetry).

Code Example:

<pre>
Line 1
    Line 2 (indented)
Line 3
</pre>

🔹 What is <code>?

The <code> tag is used to display a piece of computer code. By default, the text inside appears in a monospace font (like in programming editors).

🔹 Why use it?

  • To show inline code inside a sentence.
  • To make examples of commands or snippets stand out.
  • Often combined with <pre> to display code blocks.

🔹 Example 1 (Inline)

Code:

Use the <code>print("Hello, World!")</code> command in Python.

Live Demo:

Use the print("Hello, World!") command in Python.

🔹 Example 2 (With <pre>)

Code:

<pre>
<code>
for i in range(5):
    print(i)
</code>
</pre>

Live Demo:


for i in range(5):
    print(i)

🔹 Conclusion

✅ Use <code> for inline code. ✅ Use <pre><code> for larger code blocks.

cellpadding="10"

What it does: Adds space inside each table cell (between border and content).

Code Example:

<table border="1" cellpadding="10">
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

Live Demo:

Row 1, Cell 1 Row 1, Cell 2

Effect: Makes the table look less cramped.

Difference:

  • cellpadding → space inside the cell
  • cellspacing → space between cells

Status: Deprecated in HTML5, but works fine when CSS is not allowed.

HTML <strong> Tag

The <strong> tag is used to mark text as important. By default, browsers display it in bold, but it also adds semantic meaning (helps search engines and screen readers).

Syntax

<strong>This text is important</strong>

Example

<p>This is a normal sentence, but 
<strong>this part is very important!</strong></p>

Output

This is a normal sentence, but this part is very important!

Note

✅ Use <strong> when text is important. ❌ Use <b> only when you want bold styling without meaning.
